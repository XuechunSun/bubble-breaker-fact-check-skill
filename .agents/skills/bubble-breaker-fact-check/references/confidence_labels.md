# Bubble Breaker Confidence Labels

| Label | Meaning | Safe wording |
|---|---|---|
| confirmed | Strong evidence from primary/official/peer-reviewed sources | “证据显示…” |
| well_attested | Multiple reliable secondary sources agree | “多份资料记载…” |
| anecdotal | Based on memoirs, oral histories, later retellings, or single-source anecdotes | “一则轶事称…” |
| disputed | Credible disagreement or important uncertainty | “这个说法有争议…” |
| likely_false | Unsupported or contradicted by reliable sources | “这更像是误传…” |
